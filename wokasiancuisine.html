Wok Asian Cuisine

Wok Asian Cuisine is a fictitious business website.

This site features a Home page with general information and a link to the Menu page for quick, easy viewing of food offerings. In addition to the Menu page, there is an Order page, which utilizes Javascripting to complete calculations and provide site visitors with an order total.

In addition to these pages, there is also a Contact Us page, which features a contact form, and I've included an informational page on web security, just for fun. This site is not designed to be responsive (meaning accessible on mobile or tablet), it is for my own Javascripting skill development.

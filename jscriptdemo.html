Demonstration:
Information Security and jQuery

A JavaScript library intended to make using JavaScript easier for coders, jQuery condenses many lines of code, helping to save bandwidth, decrease page load time, and reduce data storage requirements. While there are a multitude of JavaScript libraries, jQuery’s library is robust and has become a valuable and well-respected tool in multimedia production. And while there are a multitude of instructional websites for coding utilizing JavaScript libraries, many of them are clearly written by people who already have a foundational understanding of the language. For someone new to coding with JavaScript, often the instructions are confusing.

My goal with creating a jQuery Demonstration was to create instructions that were written in plain English, using no industry or coder jargon, that would be easy for someone who is completely new to coding with JavaScript libraries. While creating the demonstration site, I kept the instructions clear and simple, incorporating examples along the way. I included references to reputable sites for further explanation for anyone who wanted a deeper dive into the subject. This allowed me to provide supplementary information while keeping the instructions simple.

Thinking through how to explain to others the concepts demonstrated on the website I created for the final project deepened my own understanding and forced me to think about what verbiage to use to simplify a relatively complex subject. This forced me to think through why the syntax is so important so that I could express the importance to site visitors. I enjoyed creating this website and helping to enhance others learning is something I’ve always enjoyed. I even worked as a tutor at a previous college. Creating this website was very reminiscent of my time as tutor.

Accessibility & Compliance-Aware Design

All web design work is informed by WCAG 2.1 accessibility standards (W3C guidelines), including readable color contrast, semantic heading structure, keyboard-accessible navigation, and clear content hierarchy.

U.S. courts have increasingly recognized websites as places of public accommodation under the ADA, leading to a rise in accessibility-related litigation. As a result, WCAG-informed design has become a widely accepted standard for reducing risk while improving usability.

Nonprofit Website Redesign (WordPress / Divi)

Performance Optimization Without a Full Rebuild

This nonprofit redesign was built on a WordPress site using the Divi theme. Instead of a full theme rebuild, we optimized the existing framework to improve accessibility, performance, and usability.

  • Image compression and resizing with Smush for faster load times, improving UX and SEO
  • WebP delivery and lazy loading for below-the-fold content, supporting smooth UX and page speed (SEO benefit)
  • Enabled Divi dynamic CSS & JavaScript to reduce unnecessary asset loading, improving UX and SEO

Technical Scores (per HubSpot Website Grader): SEO 30/30 · Mobile 30/30 · Security 10/10 · Performance 7 → 11/30 after image optimization

Demonstrates meaningful speed and performance improvements within an existing WordPress framework.

Accessibility & Compliance-Aware Design

All web design work is informed by WCAG 2.1 accessibility standards (W3C guidelines), including readable color contrast, semantic heading structure, keyboard-accessible navigation, and clear content hierarchy.

U.S. courts have increasingly recognized websites as places of public accommodation under the ADA, leading to a rise in accessibility-related litigation. As a result, WCAG-informed design has become a widely accepted standard for reducing risk while improving usability.

Full Recipe Website & Branding (HTML / CSS)

Overview: This project was built as a fully custom HTML/CSS site from the ground up, demonstrating that accessibility, performance, and cohesive branding can be achieved without a CMS.

Scope: Designed and coded the entire site, including layout, graphics, logo creation, content organization, and accessibility-focused design.

Accessibility & Usability: Implemented WCAG 2.1 standards, optimized navigation and readability, ensured semantic HTML markup, and created a cohesive brand experience from logo to content.

The site balances visual impact and performance. With rich images, scripts, and interactive features, the page size is around 4.6 MB. We optimize for speed and usability by compressing images, using modern formats, and loading content efficiently.

Technical Performance & Optimization

  • Bulk image compression & resizing: All photos are optimized to reduce file size while maintaining quality, helping the page load faster.
  • WebP image delivery: Modern browsers get smaller, high-quality images, improving load times without sacrificing visuals.
  • Lazy loading: Images that are offscreen load only when users scroll to them, reducing initial load time.
  • Minified JavaScript & CSS: Extra spaces and comments are removed from code to make files smaller and faster to load.
  • Responsive design: The site works smoothly on all devices, from mobile phones to desktops (tested with Lighthouse Mobile score: 95/100).
  • SEO best practices: The site is structured so search engines can easily understand it, helping your pages rank better (SEO score: 65/100).
  • Secure HTTPS & updated JS libraries: Protects users and ensures compatibility with modern browsers (Security score: 100/100).
  • Reduced HTTP requests: Fewer separate files are loaded, which speeds up page rendering.
  • Fast page load: The main content appears quickly for visitors (First Contentful Paint: 2.4s, Largest Contentful Paint: 6.4s).

Metrics captured using Google Lighthouse, a trusted web performance and accessibility tool built into Chrome DevTools.
